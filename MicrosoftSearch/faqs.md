# <a name="faqs"></a>Preguntas más frecuentes

Estas son las preguntas más frecuentes que tienen las organizaciones y a los usuarios acerca de la administración y el uso de Microsoft Search.
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>¿Cuál es la propuesta de valor para Microsoft Search?

Microsoft Search es una forma segura para buscar el trabajo y el contenido web. Este tipo de integración en web y la empresa sólo está disponible con Microsoft. Para obtener información relacionada con el cumplimiento de normas, vea [seguridad para la búsqueda de Microsoft](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>¿Qué características de Microsoft Search están disponibles ahora?

Para obtener una lista completa, vea [Las características de búsqueda de Microsoft](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>¿Es la descripción de consulta avanzada de soporte técnico de Microsoft Search?

Sí, Microsoft Search analiza la intención de consulta de frases más grandes. Esta característica usa AI para aprender a los usuarios habituales de frases superfluos agregan a sus consultas que no afectan a su intención de búsqueda. Por ejemplo, cuando un usuario busca 'más información sobre cómo cambiar mi contraseña póngase' se extracción las palabras menos importantes la consulta y el desencadenador según las relacionadas como 'Cambiar contraseña'.
  
Esta característica no invalidará el conjunto de palabras clave en el portal de administración.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>¿Microsoft Search busca archivos locales, así como la nube?

Microsoft Search admite la búsqueda de documentos que se encuentran en SharePoint Online, OneDrive para la empresa, así como a través de híbrido local SharePoint y SharePoint Online para los tipos de archivo más comunes de Office. Si puede buscar contenido local en SharePoint Online, debe ser capaz de encontrar contenido local con Microsoft Search. 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>¿Microsoft Search reemplaza otras experiencias de búsqueda de empresa?

Microsoft Search es una oferta complementaria que reúne los resultados de varios orígenes. Para obtener más información acerca de los orígenes y tipos de archivo admitidos, consulte las preguntas más frecuentes siguiente.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>¿Qué tipos de archivo y los orígenes de admite Microsoft Search?

Se admiten los siguientes orígenes de contenido:
  
- SharePoint Online
    
- Híbrido SharePoint (local + SPO)
    
- OneDrive para la Empresa
    
Los siguientes tipos de archivo se exponen en búsqueda de archivos y aparecen en la ficha archivos de personas y grupos:
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF

## <a name="how-does-microsoft-search-use-azure-active-directory"></a>¿Cómo utiliza Microsoft Search Azure Active Directory?

Microsoft Search usa Azure Active Directory para autenticar y autorizar el acceso a datos de su compañía. Microsoft Search se ajusta a la identidad controlar estándar recomendados en todos los productos de Microsoft. Esto significa que los usuarios se pueden firmar automáticamente a través de un inicio de sesión único.
